ACHSAM KONTROL

CARDPUTER -> ATOMS3 LITE -> TEESNY LC -> MIXPRE

Features

  • Wirerless control the Mixpre
  • Keyboard input mapping
  • ESP-NOW transmission
  • TX/RX display

Hardware

  • M5Cardputer ADV
  • ATOMS3 Lite via ESP-NOW
  • Teensy LC as USB Keyboard
